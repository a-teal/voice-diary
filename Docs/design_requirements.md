# 디자인 요구사항

Voice Diary 앱의 디자인 작업을 위한 화면 및 컴포넌트 정의입니다.

---

## 1. 주요 화면 (Screens)

### 1.1 일간 뷰 (Daily View) - 메인 화면
**경로**: `/` (홈)

**구성 요소**:
- 헤더: 날짜 표시 (< 2025년 1월 26일 일요일 >)
- 일기 카드 (있을 때)
  - 감정 이모지 (크게)
  - 키워드 태그 (최대 5개)
  - 일기 내용 텍스트 (접기/펼치기)
  - 시간 표시
- 빈 상태 (없을 때)
  - 일러스트 또는 아이콘
  - "오늘의 일기를 기록해보세요" 문구
- FAB (녹음 버튼) - 우하단

**인터랙션**:
- 좌우 스와이프로 날짜 이동
- 날짜 클릭 시 달력으로 이동

---

### 1.2 녹음 모달 (Recording Modal)
**트리거**: FAB 클릭 시 바텀시트로 등장

**구성 요소**:
- 헤더: "오늘의 일기" + 닫기 버튼
- 텍스트 영역: 실시간 음성→텍스트 표시
- 녹음 버튼
  - 대기 상태: MicPencilIcon (인디고)
  - 녹음 중: MicPencilIcon (빨간) + 펄스 애니메이션
- 녹음 시간 표시 (0:00)
- 완료 버튼 (체크 아이콘)
- 로딩 상태: "AI가 분석 중..." + 스피너

**상태**:
1. 대기 (idle)
2. 녹음 중 (recording)
3. AI 분석 중 (analyzing)
4. 완료/에러

**레이아웃 (2025-01-30 업데이트)**:
- 컴팩트 디자인 (min-height 제거)
- 버튼이 컨텐츠 바로 아래 위치 (mt-auto 제거)
- 아이콘 크기: 16x16 (idle), 20x20 (recording)
- 버튼 패딩: py-3.5

---

### 1.3 월간 캘린더 뷰 (Calendar View)
**경로**: `/calendar`

**구성 요소**:
- 헤더: 월 표시 (< 2025년 1월 >)
- 요일 표시 (일~토)
- 날짜 그리드 (7x6)
  - 오늘 날짜 하이라이트
  - 일기 있는 날: 감정 이모지 표시
  - 일기 없는 날: 숫자만

**인터랙션**:
- 날짜 클릭 → 일간 뷰로 이동
- 좌우 스와이프/화살표로 월 이동

---

### 1.4 통계 뷰 (Stats View)
**경로**: `/stats`

**구성 요소**:
- 기간 선택 탭: 7일 / 30일
- 감정 변화 그래프 (라인 차트)
  - X축: 날짜
  - Y축: 감정 (긍정↔부정)
  - 데이터 포인트에 이모지 표시
- 키워드 클라우드
  - 빈도에 따른 크기/색상 차이
- 요약 카드
  - 총 기록 일수
  - 가장 많은 감정
  - 가장 많은 키워드

---

### 1.5 하단 네비게이션 (Bottom Navigation)
**위치**: 모든 화면 하단 고정

**탭 구성**:
1. 일간 (홈 아이콘)
2. 월간 (캘린더 아이콘)
3. 통계 (차트 아이콘)

**상태**:
- 활성: 강조 색상 + 라벨
- 비활성: 회색

---

## 2. 컴포넌트 (Components)

## 4. 아이콘 필요 목록

- 마이크+연필 (MicPencilIcon) - 커스텀 브랜드 아이콘 ✅
- 정지 (녹음 중지)
- 체크 (완료)
- X (닫기)
- 홈
- 캘린더
- 차트/통계
- 화살표 좌/우
- 펼치기/접기

### 4.1 브랜드 아이콘 (MicPencilIcon)
- **위치**: `components/icons/MicPencilIcon.tsx`
- **컨셉**: 마이크 + 연필 결합 (음성 + 기록)
- **사용처**:
  - FAB 녹음 버튼
  - 녹음 모달 아이콘
  - 앱 아이콘 (192x192, 512x512)
- **색상**: currentColor (부모 요소의 text 색상 상속)
- **배경**: 검정 (#000000) - 앱 아이콘용

## 10. 다국어 지원 (i18n)

### 지원 언어
- 영어 (en) - 기본값
- 한국어 (ko)

### 작동 방식
1. 브라우저 언어 자동 감지 (`navigator.language`)
2. 지원 언어 있으면 해당 언어 표시
3. 없으면 영어 (fallback)

### 번역 파일 위치
```
apps/web/messages/
├── en.json
└── ko.json
```
