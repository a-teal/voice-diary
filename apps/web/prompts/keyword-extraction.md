# HASHTAG_RULES

Voice Diary - 키워드 해시태그 규칙 파일

---

## 1. 목적

* 음성 일기에서 **검색·회고·분류**에 유용한 키워드만 해시태그로 추출한다.
* 감정·사건·주제를 **과도한 중복 없이** 구조화한다.
* 나중의 내가 다시 봤을 때 맥락을 빠르게 떠올릴 수 있도록 한다.

---

## 2. 기본 생성 규칙

* 해시태그 개수: **3-6개**
* 표기 규칙:
  * 모두 소문자
  * 공백 없음
  * 특수문자 제거
  * 한글 또는 영문 단어만 사용
* 형식 예시:
  * `#회의`, `#불안`, `#work`

---

## 3. 우선순위 카테고리

아래 순서를 기준으로 선택한다. (모든 카테고리를 반드시 채울 필요는 없음)

### 1) 핵심 주제 (필수, 1-2개)

* 오늘의 사건, 행동, 상황을 가장 잘 대표하는 키워드
* 예시:
  * `#회의`, `#운동`, `#아이`, `#여행`, `#공부`

### 2) 감정 상태 (선택, 1-2개)

* 음성에서 **명확히 드러난 감정만** 사용
* 추론이나 과도한 해석 금지
* 예시:
  * `#불안`, `#뿌듯`, `#지침`, `#설렘`, `#우울`

### 3) 맥락/영역 (선택, 0-1개)

* 삶의 영역 단위 태그
* 예시:
  * `#work`, `#family`, `#health`, `#money`, `#relationship`

### 4) 의미 있는 키워드 (선택, 0-1개)

* 반복적으로 등장하며 회고 시 의미를 가지는 단어
* 예시:
  * `#결정`, `#갈등`, `#도전`, `#휴식`

---

## 4. 금지 규칙

* 문장형 해시태그
  * 예: `#오늘너무힘들었다`
* 지나치게 일반적인 단어
  * 예: `#하루`, `#일상`, `#기록`
* 동일 의미의 중복 태그
  * 예: `#기쁨` + `#행복`
* 음성에 언급되지 않은 감정·의도 추가

---

## 5. 정규화 규칙 (Normalization)

* 유사 의미는 **대표 태그 하나로 통일**한다.
  * `#회사일`, `#업무` → `#work`
  * `#짜증`, `#화남` → `#분노`
* 복합 개념은 **짧은 핵심 명사**로 축약한다.
  * `#중요한결정해야함` → `#결정`

---

## 6. 예시

### 입력 음성

> "오늘 회의가 너무 길어서 지쳤다. 그래도 방향은 잡힌 것 같아."

### 출력 해시태그

```
#회의 #지침 #work #결정
```

---

## 7. 톤 가이드

* 분석적이기보다 **일기 친구 같은 느낌**을 유지한다.
* 나중의 내가 봤을 때 **상황과 감정이 바로 떠오르는 수준**을 목표로 한다.
